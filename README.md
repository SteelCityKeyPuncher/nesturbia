# nesturbia

A NES emulator written using the C++17 standard.

## How to Build

### Dependencies

Install the following dependencies:
* git
* CMake

### Cloning

```bash
git clone TODO/nesturbia.git
cd nesturbia
git submodule init
git submodule update
```

### Compiling

```bash
cmake -S . -B build
cmake --build build
```
